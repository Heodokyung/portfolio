Finnq App 2.0/3.0 개편 서비스 운영

Description

Three Points

  • 공통 UI모션 개발
  • 차트 API 구현
  • SCSS, GIT, CSS3
  • 참여도 :
    100%
  • 작업환경 : HTML5, CSS3, Jquery, Sass, Zeplin, Git/SVN, Jira, Wiki Confluence, Slack
  • 기간 : 2018.08.01 - 재직중
  • 담당업무 :
    • Finnq App 서비스 운영
    • 신규 추가 서비스 퍼블리싱 및 이벤트 페이지 GUI구현
    • Ajax 데이터 연동 이벤트 구현
    • 차트 구현
    • 트러블 슈팅(trouble shooting) 및 단위테스트
    • Jenkins 및 ChangeFlow, Git을 통한 형상 배포
  • 운영 서비스 2.0
    • 핀크계좌 : 무료송금과 하나은행 ATM출금, 캐시백 혜택까지 받는 입출금 통장 서비스
    • 소비평가 : 자신이 사용한 영수증으로 하는 소비평가 기능 제공
    • AI핀고분석 : 나의 소비, 소득을 분석하고 알맞은 금융 생활 조언 서비스
    • 핀크마켓 : 혜택 많은 금융상품과 최고 할인률 기프티콘몰 서비스 제공
    • 기타 이벤트 페이지 구현
  • 운영 서비스 3.0
    • 대출 : T통신스코어를 활용하여 자신에게 최적화 된 금리비교, 추천상품을 제공
    • 금융상품 : 예적금, 카드, 대출, 투자(p2p, 아트투자), 보험, 생활 속 핀크(해외송금, 기프티콘몰)등 일상속에서 자신에게 꼭 필요한 금융서비스를 제공
    • 리얼리 : SNS기반 금융 커뮤니케이션으로 익명(별명)으로 매달 주어지는 챌리저를 도전하고 타인의 금융을 알아보는 서비스
    • 기타 이벤트 페이지 구현
  • 성과 :
    • Semantic Markup 퍼블리싱
    • 고해상력(High PPI) 디스플레이 대응
    • Android/IOS 앱 오픈
    • 1.0에서 2.0/3.0으로 개편하면서 UI/UX 향상
    • 오픈뱅킹API 도입으로 은행/증권사 연결 다양화
    • UI 개편으로 사용성 개선, 신규회원 유입 증가
    • 신규 서비스 추가 런칭
    • 송금 서비스 전면 개편
    • 금융 서비스 전면 개편
    • SNS기반 금융서비스 리얼리런칭
